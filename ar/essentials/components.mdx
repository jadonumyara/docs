تتيح لك مكوّنات Mintlify بناء صفحات وثائق غنية وتفاعلية باستخدام MDX. فيما يلي بعض المكوّنات الشائعة الاستخدام.

اطلع على [نظرة عامة على المكوّنات](https://mintlify.com/docs/components) في وثائق Mintlify للحصول على القائمة الكاملة.

## مربعات التنبيه

استخدم مربعات التنبيه لجذب الانتباه إلى المعلومات المهمة.

<Tip>تُبرز **النصيحة** اختصارًا مفيدًا أو أفضل ممارسة.</Tip>

<Warning>يشير **التحذير** إلى شيء قد يسبّب مشاكل.</Warning>

<Info>يوفّر مربع **معلومات** سياقًا تكميليًا.</Info>

```mdx
<Note>A **Note** is for general information that helps the reader.</Note>

<Tip>A **Tip** highlights a helpful shortcut or best practice.</Tip>

<Warning>A **Warning** flags something that could cause problems.</Warning>

<Info>An **Info** callout provides supplementary context.</Info>
```

## الخطوات

تعرض الخطوات الإجراءات المرقّمة بوضوح.

<Steps>
  <Step title="تثبيت الحزمة">
    ```bash
    npm install my-package
    ```
  </Step>
  <Step title="التهيئة">
    نفّذ أمر init في جذر مشروعك.
    ```bash
    npx my-package init
    ```
  </Step>
  <Step title="أنت جاهز">
    ابدأ باستخدام الحزمة في الكود الخاص بك.
  </Step>
</Steps>

````mdx
<Steps>
  <Step title="Install the package">
    ```bash
    npm install my-package
    ```
  </Step>
  <Step title="Initialize">
    Run the init command in your project root.
  </Step>
  <Step title="You're ready">
    Start using the package in your code.
  </Step>
</Steps>
````

## مجموعات الكود

يعرض `CodeGroup` عدة مقتطفات كود كعلامات تبويب — مثالي لعرض المثال نفسه بعدة لغات.

<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('https://api.example.com/users');
  const users = await response.json();
  ```

  ```python Python
  import requests
  response = requests.get('https://api.example.com/users')
  users = response.json()
  ```

  ```bash curl
  curl https://api.example.com/users
  ```
</CodeGroup>

````mdx
<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('https://api.example.com/users');
  ```

  ```python Python
  response = requests.get('https://api.example.com/users')
  ```
</CodeGroup>
````
