شريط الأدوات السياقي هو شريط إجراءات عائم يظهر على كل صفحة من وثائقك. يتيح للقرّاء نسخ محتوى الصفحة على الفور أو فتحها في مساعد ذكاء اصطناعي أو إضافة وثائقك كخادم MCP — دون مغادرة الصفحة.

كوّنه في `docs.json` ضمن `contextual.options`:

```json docs.json
{
  "contextual": {
    "options": ["copy", "view", "claude", "chatgpt", "mcp", "cursor", "vscode"]
  }
}
```

## الخيارات المتاحة

### إجراءات المحتوى

| الخيار | الوصف |
|--------|-------------|
| `copy` | ينسخ الصفحة بصيغة Markdown إلى الحافظة |
| `view` | يفتح إصدار Markdown الخام للصفحة (عنوان `.md`) |

### مساعدو الذكاء الاصطناعي

| الخيار | الوصف |
|--------|-------------|
| `claude` | يفتح الصفحة في Claude |
| `chatgpt` | يفتح الصفحة في ChatGPT |
| `perplexity` | يفتح الصفحة في Perplexity |
| `grok` | يفتح الصفحة في Grok |
| `aistudio` | يفتح الصفحة في Google AI Studio |

### تكاملات المحرّر

| الخيار | الوصف |
|--------|-------------|
| `cursor` | يضيف الصفحة إلى سياق Cursor |
| `vscode` | يضيف الصفحة إلى VS Code Copilot |
| `windsurf` | يضيف الصفحة إلى Cascade الخاص بـ Windsurf |
| `devin` | يفتح الصفحة في Devin |

### MCP

| الخيار | الوصف |
|--------|-------------|
| `mcp` | يعرض تعليمات لاستخدام وثائقك كخادم MCP |
| `add-mcp` | يوفّر مطالبة بنقرة واحدة لإضافة خادم MCP لوثائقك إلى أداة ذكاء اصطناعي |

<Tip>
  خيارا `mcp` و`add-mcp` فعّالان بشكل خاص — يتيحان للمطوّرين توصيل موقع وثائقك بالكامل بأي مساعد ذكاء اصطناعي متوافق مع MCP، حتى يتمكّن من الإجابة على الأسئلة المتعلّقة بمنتجك مباشرة.
</Tip>

## الخيارات المخصّصة

يمكنك إضافة إدخالات مخصّصة إلى شريط الأدوات تحتوي على عنوان ووصف وأيقونة ورابط. يدعم `href` عناصر نائبة ديناميكية:

| العنصر النائب | يتحوّل إلى |
|-------------|-------------|
| `$page` | عنوان URL الكامل للصفحة الحالية |
| `$path` | مسار الصفحة الحالية |
| `$mcp` | عنوان URL لخادم MCP لوثائقك |

```json docs.json
{
  "contextual": {
    "options": [
      "copy",
      "claude",
      {
        "title": "Open issue",
        "description": "Report a problem with this page",
        "icon": "github",
        "href": "https://github.com/your-org/docs/issues/new?title=Issue+on+$path"
      }
    ]
  }
}
```

<Note>
  شريط الأدوات السياقي مرئي فقط على عمليات النشر للمعاينة والإنتاج، وليس في المعاينة المحلية `mint dev`.
</Note>

## الإقران مع Visibility

يكشف خيارا `copy` و`view` في شريط الأدوات عن محتوى Markdown لصفحتك مباشرة. إذا كنت تستخدم مكوّن [`Visibility`](/ar/agent-ready/visibility)، فسيتم تضمين كتل `for="agents"` في تلك Markdown — مما يمنح مساعدي الذكاء الاصطناعي سياقًا منظّمًا لا يراه القرّاء من البشر.
